Local Storage

Local Storage is a web API that allows web applications to store data locally in the user's browser. Unlike session storage, data stored in localStorage persists even after the browser is closed, making it ideal for saving user preferences, form data, and application state.

The localStorage API provides simple methods to store and retrieve key-value pairs. Data is stored as strings, so when storing objects or numbers, we use JSON.stringify() and JSON.parse() to convert between formats. The storage limit is 5-10MB per domain.

Local Storage in This Website

On this website, we track the number of times a "contact us" form is submitted. Here's the code that does that: 

const contactForm = document.querySelector('form');

if (contactForm) {
    contactForm.addEventListener('submit', () => {
        let count = parseInt(localStorage.getItem('number_of_contact_messages')) || 0;
        count++;
        localStorage.setItem('number_of_contact_messages', count);
    });
}

const numberOfContactMessages = document.getElementById('number_of_contact_messages');
if (numberOfContactMessages) {
    numberOfContactMessages.textContent = localStorage.getItem('number_of_contact_messages') || 0;
}
